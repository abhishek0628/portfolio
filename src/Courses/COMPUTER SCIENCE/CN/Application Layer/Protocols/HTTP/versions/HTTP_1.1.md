HTTP/1.1 is a text-based, request–response protocol used between:

Client (browser)
Server (web server)

characteristics:
1. Persistent Connections
2. Request Pipelining(head of line blocking)
3. Host Header
4. Status Codes Standardized
5. Chunked Transfer Encoding
6. Better caching control
7. Improved connection management

Drawbacks:
1. Head-of-Line Blocking
2. Multiple connections needed
3. Text-based overhead
4. No real parallelism(response in serial order)
